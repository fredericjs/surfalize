========
Examples
========

This section collects worked examples as Jupyter notebooks. Each notebook is
rendered with its stored outputs, so you can read through a complete analysis
without running anything yourself.

To add a new example, drop a ``.ipynb`` file into ``docs/source/examples/`` and
list it in the toctree below.
